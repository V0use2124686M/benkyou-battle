# べんきょうバトル

小学2〜3年生向けの学習ゲーム。問題に正解してモンスターを攻撃するバトル形式。
`index.html` 1ファイルで完結し、ブラウザで開くだけで動く(ビルド・サーバー・通信不要)。

## モード
- **かけ算モード**: 九九(1×1〜9×9)からランダム出題
- **わり算モード**: 答えが整数になる割り算(九九の逆算)からランダム出題
- **かんじモード**: 小2・小3の配当漢字約120問の読みを4択で出題

## 遊び方
スマホまたはPCのブラウザで `index.html` を開くだけ。

## モンスター画像の差し替え
`index.html` と同じフォルダに `monster1.png` 〜 `monster12.png` を置くと、
絵文字の代わりにその画像が表示される(無ければ絵文字にフォールバック)。

## 記録
モードごとのハイスコアを localStorage に保存する。
