# べんきょうバトル

小学2〜3年生向けの学習ゲーム。問題に正解してモンスターを攻撃するバトル形式。
`index.html` 1ファイルで完結し、ブラウザで開くだけで動く(ビルド・サーバー・通信不要)。

## モード
- **かけ算モード**: 九九からランダム出題。レベル7以降は2桁×1桁も混ざり、レベル9以降はさらに範囲が広がる
- **わり算モード**: 答えが整数になる割り算。レベル7以降は2桁÷1桁(答えも2桁、割られる数は99まで)も混ざる
- **かんじモード**: 配当漢字216問(1〜3文字の熟語を含む)を4択で出題。「漢字→読み」と「読み→漢字」の2形式が半々で出る。レベル1〜6は小2中心、7〜8は小3中心、9以降は小3+小4。送り仮名のある言葉(泣く・選ぶ等)は選択肢も送り仮名ありの言葉だけになる
- **ちりモード**: 4種類の問題がランダムに混ざって出題される
  - 都道府県⇔県庁所在地(40%): レベル1〜6はよく知られた16都道府県、7〜8は中堅16県、9以降は残り15県を加えた31県から出題
  - 何地方の都道府県か(20%): 8地方区分(北海道・東北・関東・中部・近畿・中国・四国・九州)から選ぶ
  - 地図記号(20%): 学校・郵便局・神社・寺・病院など16種類をSVG/絵文字アイコンで出題。国土地理院の記号の由来(例: 田=稲を刈った跡、畑=植物の二葉、病院=十字、警察署=交番の記号を丸で囲む)にもとづいた簡略デザイン。レベルに応じて記号の難度も上がる
  - 世界の国当て(20%): ヒントの文章から国名を当てる。有名な30か国を出題

## 難易度とクリア
- レベル(=モンスターの番号)が上がるほど問題が難しくなる。3段階: レベル1〜6 / 7〜8 / 9以降
- かけ算・わり算はレベル9以降、約35%の確率で文章題が出る
- **レベル15のモンスターを倒すとゲームクリア**(クリア画面が出る)
- バトル中は「やめる」ボタンでいつでも中断でき、そこまでのスコアで結果画面になる

## 遊び方
スマホまたはPCのブラウザで `index.html` を開くだけ。

## モンスター画像
モンスターはレベルごとに1体ずつ全15体で、`monster01.png` 〜 `monster15.png` を表示する
(レベル1〜9の元画像はルートの `picture/Level1〜9.png`)。
レベル10以降はボス級の新キャラで、ステータスも強化される。
画像が無い場合は絵文字にフォールバックする。差し替えるときは同名で上書きするだけでよい。

## 記録
モードごとのハイスコアを localStorage に保存する(端末×ブラウザごと。サーバーには何も送らない)。
タイトル画面の「スコアを リセット」ボタン(2回押して確定)で全モードのハイスコアを消せる。
